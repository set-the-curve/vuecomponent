    # Card

    Hello World

    I am a card component. I can be used to group content for discrete items, along with images or actions. I also come with a shadow by default to make my content stand out.

    # Basic Info

    A Vue component in the tradional "Card" style. Can be used with optional title or action buttons.

    # Props

    Prop Type Description
    width String Width of the card. Accepts any standard CSS format (px, rem)
    title String Optional header to use at top of card.

    # Example

    <Card width="400px" title="Hello World">
  <p>I am a card component.  I can be used to group content for discrete items, along with images or actions.  I also come with a shadow by default to make my content stand out.</p>
</Card>

    # Source

    <template>
  <div class="card" v-bind:style="{ width: width }">
    <h3>{{title}}</h3>
    <div class="card-body">
      <slot></slot>
    </div>
    <div v-if="false" class="card-actions">
    </div>
  </div>
</template>

<script>
export default {
  name: 'Card',
  props: {
    title: String,
    width: String,
  }
}
</script>
